Strings

Strings

Inicialización

Inicialización

  • Los strings se inicializan con comillas simples, dobles o backticks(comillas francesas).

    let a = 'Hola'; let b = "Hola"; let c = `Hola`;
Métodos

Métodos

  • length

    length() devuelve la longitud del string.

    let a = 'Hola'; console.log('La longitud de a es: ' + a.length); //La longitud de a es: 4

  • toUpperCase

    toUpperCase() devuelve el string en mayúsculas.

    let a = 'Hola'; console.log(a.toUpperCase()); //HOLA

  • toLowerCase

    toLowerCase() devuelve el string en minúsculas.

    let a = 'HOLA'; console.log(a.toLowerCase()); //hola

  • indexOf

    indexOf() devuelve la posición de la primera aparición del string indicado.

    Se le puede pasar un segundo parámetro para indicar desde qué posición empezar a buscar.

    Si no encuentra el string, devuelve -1.

    let a = 'Hola Mundo'; console.log(a.indexOf('o')); //1 console.log(a.indexOf('o', 2)); //9 console.log(a.indexOf('z')); //-1

  • lastIndexOf

    lastIndexOf() devuelve la posición de la última aparición del string indicado.

    Se le puede pasar un segundo parámetro para indicar antes de qué posición empezar a buscar.

    Si no encuentra el string, devuelve -1.

    let a = 'Hola Mundo'; console.log(a.lastIndexOf('o')); //9 console.log(a.lastIndexOf('o', 8)); //1 console.log(a.lastIndexOf('z')); //-1

  • includes

    includes() devuelve true si el string contiene el string indicado.

    Se le puede pasar un segundo parámetro para indicar desde qué posición empezar a buscar.

    let a = 'Hola Mundo'; console.log(a.includes('Mundo')); //true console.log(a.includes('Mundo', 5)); //false

  • startsWith

    startsWith() devuelve true si el string empieza con el string indicado.

    Se le puede pasar un segundo parámetro para indicar desde qué posición empezar a buscar.

    let a = 'Hola Mundo'; console.log(a.startsWith('Hola')); //true console.log(a.startsWith('Mundo', 5)); //true

  • endsWith

    endsWith() devuelve true si el string termina con el string indicado.

    Se le puede pasar un segundo parámetro para indicar antes de qué posición empezar a buscar.

    let a = 'Hola Mundo'; console.log(a.endsWith('Mundo')); //true console.log(a.endsWith('Hola', 4)); //true

  • repeat

    repeat() devuelve un nuevo string con el string selecionado repetido el número de veces indicado.

    let a = 'Hola'; console.log(a.repeat(3)); //HolaHolaHola

  • replace

    replace() devuelve un nuevo string con el string selecionado reemplazado por el string indicado.

    let a = 'Hola Mundo'; console.log(a.replace('Mundo', 'Mundo!')); //Hola Mundo! console.log(a.replace('o', 'u')); //Hula Mundo

  • replaceAll

    replaceAll() devuelve un nuevo string con el string selecionado reemplazado por el string indicado en todas las apariciones.

    let a = 'Hola Mundo'; console.log(a.replaceAll('o', 'u')); //Hula Mundu

  • split

    split() devuelve un array con los elementos del string separados por el string indicado.

    Se le puede pasar un segundo parámetro para indicar el número máximo de elementos que se devuelven.

    let a = 'Hola Mundo'; console.log(a.split(' ')); //['Hola', 'Mundo'] console.log(a.split(' ', 1)); //['Hola']

  • slice

    slice() devuelve el string desde la primera posición indicada hasta la segunda, sin incluirla.

    Si no se indica la posición final, devuelve el string desde la posición indicada hasta el final.

    let a = 'Hola Mundo'; console.log(a.slice(5)); //Mundo console.log(a.slice(5, 8)); //Mun

  • trim

    trim() devuelve un nuevo string sin espacios al principio ni al final.

    trimStart() devuelve un nuevo string sin espacios al principio.

    trimEnd() devuelve un nuevo string sin espacios al final.

    let a = ' Hola Mundo '; let b = a.trim(); let c = a.trimStart(); let d = a.trimEnd(); console.log(`¿¿${b}??`); // ¿¿Hola Mundo?? console.log(`¿¿${c}??`); // ¿¿Hola Mundo ?? console.log(`¿¿${d}??`); // ¿¿ Hola Mundo??

  • concat

    concat() devuelve un nuevo string con el string indicado añadido al final.

    let a = 'Hola'; console.log(a.concat(' Mundo')); //Hola Mundo
Operadores

Operadores

Aritméticos

Operadores aritméticos

  • Suma

    Se utiliza para sumar dos valores.

    Se representa con el símbolo +.

    let a = 12; let b = 10; let result = a+b; //22

  • Resta

    Se utiliza para restar dos valores.

    Se representa con el símbolo -.

    let a = 12; let b = 10; let result = a-b; //2

  • Multiplicación

    Se utiliza para multiplicar dos valores.

    Se representa con el símbolo *.

    let a = 12; let b = 10; let result = a*b; //120

  • División

    Se utiliza para dividir dos valores.

    Se representa con el símbolo /.

    let a = 12; let b = 10; let result = a/b; //1.2

  • Módulo

    Se utiliza para obtener el resto entero de la división de dos valores.

    Se representa con el símbolo %.

    Se puede utilizar para comprobar si un número es par.

    let a = 5; let b = 2; let result = a%b; //1 let a = 6; let b = 2; let result = a%b; //0 (es par)

  • Exponenciación

    Se utiliza para elevar un valor a una potencia.

    Se representa con el símbolo **.

    let a = 5; let b = 2; let result = a**b; //25

  • Incremento

    Se utiliza para incrementar en 1 el valor de una variable.

    Se representa con el símbolo ++.

    let a = 5; console.log(++a); //6 (se incrementa antes de mostrar el valor) console.log(a++); //6 (se incrementa después de mostrar el valor) console.log(a); //7 (se muestra el valor incrementado)

  • Decremento

    Se utiliza para decrementar en 1 el valor de una variable.

    Se representa con el símbolo --.

    let a = 5; console.log(--a); //4 (se decrementa antes de mostrar el valor) console.log(a--); //4 (se decrementa después de mostrar el valor) console.log(a); //3 (se muestra el valor decrementado)

  • Unario

    Se utiliza para convertir un valor en un número si aún no lo es.

    Se representa con el símbolo + o -.

    let a = "5"; let b = "5 años" let c = +a; //5 let d = -a; //-5 let e = +b; //NaN console.log(typeof a); //string console.log(typeof b); //string console.log(typeof c); //number console.log(typeof d); //number console.log(typeof e); //number
Asignación

Operadores de asignación

  • Asignación simple

    Se utiliza para asignar un valor a una variable.

    Se representa con el símbolo =.

    let a = 5;

  • Asignación de suma

    Se utiliza para sumar un valor a una variable.

    Se representa con el símbolo +=.

    let a = 5; let b = 2; a = a + b; //7 a += b; //9

  • Asignación de resta

    Se utiliza para restar un valor a una variable.

    Se representa con el símbolo -=.

    let a = 5; let b = 2; a = a - b; //3 a -= b; //1

  • Asignación de multiplicación

    Se utiliza para multiplicar un valor a una variable.

    Se representa con el símbolo *=.

    let a = 5; let b = 2; a = a * b; //10 a *= b; //20

  • Asignación de división

    Se utiliza para dividir un valor a una variable.

    Se representa con el símbolo /=.

    let a = 5; let b = 2; a = a / b; //2.5 a /= b; //1.25

  • Asignación de módulo

    Se utiliza para obtener el resto entero de la división de un valor a una variable.

    Se representa con el símbolo %=.

    let a = 5; let b = 2; a %= b; //1

  • Asignación de exponenciación

    Se utiliza para elevar un valor a una potencia a una variable.

    Se representa con el símbolo **=.

    let a = 5; let b = 2; a **= b; //25
Comparación

Operadores de Comparación

  • Igualdad

    Se utiliza para comparar si dos valores son iguales.

    Se representa con el símbolo ==.

    El operador de igualdad convierte los valores a un mismo tipo antes de compararlos.

    let a = 5; let b = "5"; console.log(a == b); //true console.log(typeof a); //number console.log(typeof b); //string

  • Desigualdad

    Se utiliza para comparar si dos valores son diferentes.

    Se representa con el símbolo !=.

    El operador de desigualdad convierte los valores a un mismo tipo antes de compararlos.

    let a = 5; let b = "5"; console.log(a != b); //false console.log(typeof a); //number console.log(typeof b); //string

  • Estrictamente igual

    Se utiliza para comparar si dos valores son iguales y del mismo tipo.

    Se representa con el símbolo ===.

    El operador de igualdad estricta no convierte los valores a un mismo tipo antes de compararlos.

    let a = 5; let b = "5"; console.log(a === b); //false

  • Estrictamente desigual

    Se utiliza para comparar si dos valores son diferentes y/o de diferente tipo.

    Se representa con el símbolo !==.

    El operador de desigualdad estricta no convierte los valores a un mismo tipo antes de compararlos.

    let a = 5; let b = "5"; console.log(a !== b); //true

    • Mayor que

    Se utiliza para comparar si un valor es mayor que otro.

    Se representa con el símbolo >.

    let a = 5; let b = 2; console.log(a > b); //true

  • Mayor o igual que

    Se utiliza para comparar si un valor es mayor o igual que otro.

    Se representa con el símbolo >=.

    let a = 5; let b = 2; console.log(a >= b); //true

  • Menor que

    Se utiliza para comparar si un valor es menor que otro.

    Se representa con el símbolo <.

    let a = 5; let b = 2; console.log(a < b); //false

  • Menor o igual que

    Se utiliza para comparar si un valor es menor o igual que otro.

    Se representa con el símbolo <=.

    let a = 5; let b = 2; console.log(a <= b); //false
Lógicos

Operadores Lógicos

  • AND

    Se utiliza para evaluar si dos o más expresiones son verdaderas.

    Se representa con el símbolo &&.

    Si una de las expresiones es falsa, el resultado será falso.

    Si todas las expresiones son verdaderas, el resultado será verdadero.

    Si una de las expresiones es falsa, el resto de las expresiones posteriores no se evalúan.

    Se puede usar para devolver un valor alternativo.

    let a = 5; let b = 2; let c console.log(a > 0 && b > 0); //true console.log(a > 0 && b < 0); //false console.log(a<0 && b > 0); //false console.log(a && 'Hola'); //Hola console.log(a && c); //undefined console.log(a<0 && c); //false

  • OR

    Se utiliza para evaluar si dos o más expresiones son verdaderas.

    Se representa con el símbolo ||.

    Si una de las expresiones es verdadera, el resultado será verdadero.

    Si todas las expresiones son falsas, el resultado será falso.

    Si una de las expresiones es verdadera, el resto de las expresiones posteriores no se evalúan.

    Se puede usar para devolver un valor alternativo.

    let a = 5; let b = 2; let c console.log(a > 0 || b > 0); //true console.log(a > 0 || b < 0); //true console.log(a<0 || b > 0); //true console.log(a || 'Hola'); //5 console.log(c || 'Hola'); //Hola

  • NOT

    Se utiliza para invertir el valor de una expresión.

    Se representa con el símbolo !.

    Si la expresión es verdadera, el resultado será falso.

    Si la expresión es falsa, el resultado será verdadero.

    let a = 5; let b = 2; console.log(!(a>b)); //false console.log(!(a=b)); //true

  • ??

    Se llama Nullish Coalescing(Coalescente nulo).

    Se utiliza para devolver el primer valor que no sea null o undefined.

    Se representa con el símbolo ??.

    Si el primer valor es null o undefined, el resultado será el segundo valor.

    Si el primer valor no es null o undefined, el resultado será el primer valor.

    let a = 5; let b = 2; let c console.log(a ?? b); //5 console.log(c ?? b); //2
Cadena

Operadores de Cadena

  • Concatenación

    Se utiliza para unir dos cadenas de texto.

    Se representa con el símbolo +.

    let a = "Hola"; let b = "Mundo"; let c = '3'; let d = 4; let result = a + b; //"HolaMundo" let result2 = c + d; //"34"

  • Asignación de concatenación

    Se utiliza para unir dos cadenas de texto a una variable.

    Se representa con el símbolo +=.

    let a = "Hola"; let b = "Mundo"; let c = '3'; let d = 4; a += b; //"HolaMundo" c += d; //"34"
Ternario

Operadores de Condición Ternaria

  • Condición Ternaria

    Se utiliza para evaluar una condición y devolver un valor.

    Se representa con el símbolo ? y :.

    let a = 5; let b = 2; let result = a > b ? "a es mayor que b" : "a es menor que b"; //"a es mayor que b"
Delete

Operador Delete

  • Delete

    Se utiliza para eliminar una propiedad de un objeto.

    Se representa con la palabra delete.

    let person = { name: "John", age: 30, city: "New York" }; delete person.age; console.log(person); //{name: "John", city: "New York"} console.log(person.age); //undefined console.log(delete person.city); //true
Condicionales

Condicionales

if, else if, else

if, else if, else

  • if

    Se utiliza para ejecutar un bloque de código si una condición es verdadera.

    Se representa con la palabra if.

    let a = 5; let b = 2; if (a > b) { console.log("a es mayor que b"); } a=2;

  • if, else

    Se utiliza para ejecutar un bloque de código si una condición es verdadera, y otro bloque de código si la condición es falsa.

    Se representa con las palabras if y else.

    let a = 5; let b = 2; if (a > b) { console.log("a es mayor que b"); } else { console.log("a es menor que b"); }

  • if, else if, else

    Se utiliza para ejecutar un bloque de código en función de múltiples condiciones.

    Se representa con las palabras if, else if y else.

    Se pueden encadenar tantos else if como se necesiten.

    let edad = 18; if (edad < 18) { console.log("Eres menor de edad"); } else if (edad >= 18 && edad < 65) { console.log("Eres mayor de edad"); } else { console.log("Eres jubilado"); } if (edad < 18) { console.log("Eres menor de edad"); } if (edad >= 18 && edad < 65) { console.log("Eres mayor de edad"); } if (edad >= 65) { console.log("Eres jubilado"); }
switch

switch

  • switch

    Se utiliza para ejecutar un bloque de código en función de varios casos.

    Se representa con la palabra switch.

    Se pueden encadenar tantos case como se necesiten.

    Se puede utilizar la palabra default para ejecutar un bloque de código si ninguna de las condiciones se cumple.

    Se ultiliza la palabra break para romper cada caso para que no siga ejecutando las instrucciones.

    Si usamos switch dentro de una función, se puede sustituir la palabra break por return.

    let edad = 18; switch (edad) { case 18: console.log("Eres mayor de edad"); break; //Importante romper cada caso para que no siga ejecutando las instrucciones case 65: console.log("Eres jubilado"); break; default: console.log("No se ha encontrado ninguna coincidencia"); }
